Pipeline de Procesamiento de Datos

1. Recolección de Gestos

Captura de imágenes de gestos del lenguaje de señas (A-Z)

2. Limpieza de Datos

Eliminación de imágenes borrosas, mal iluminadas o con gestos incorrectos

3. Preprocesamiento

Redimensionamiento a 128x128px, normalización y conversión a escala de grises

4. Data Augmentation

Rotación, zoom, flip horizontal para aumentar la variabilidad del dataset

5. Etiquetado de Letras

Asignación de letras (A-Z) correspondientes a cada gesto de seña

6. División del Dataset

Separación en conjuntos de entrenamiento (70%), validación (15%) y prueba (15%)

Normalización

Escalado de valores de píxeles al rango [0,1] o [-1,1]

Feature Extraction

Extracción de características visuales relevantes

Balanceo de Clases

Técnicas SMOTE y undersampling para equilibrar categorías
