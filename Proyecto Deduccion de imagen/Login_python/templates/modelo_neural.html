¿Cómo Funciona el Reconocimiento?

El sistema identifica letras del alfabeto de señas mediante inteligencia artificial

1. Captura

Sube una imagen o usa tu webcam para capturar el gesto de seña

2. Análisis

La red neuronal procesa y analiza el gesto en menos de 1 segundo

3. Reconocimiento

Identifica la letra correspondiente del alfabeto (A-Z)

4. Resultado

Muestra la letra predicha con nivel de confianza
